# Güerometry Dash

Archivos listos para GitHub Pages.

## Subir a GitHub Pages
1. Crea un repositorio nuevo.
2. Sube `index.html`, `stereo.mp3`, `cantletgo.mp3`, `toe2.mp3`.
3. Ve a Settings → Pages.
4. En Branch elige `main` y carpeta `/root`.
5. Copia la URL que te dé GitHub Pages.
6. En MIT App Inventor pon un WebViewer con esa URL en `HomeUrl`.

## Controles
- Click / toque / espacio = saltar.
- Botón ☰ = pausa.
